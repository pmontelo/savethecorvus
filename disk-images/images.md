# Apple II Disk Image Files

### ProROM 2025 Disk 1 - ProROM Configuration Utility - DOS 3.3
Disk to configure the size of the hard disk drive(s) and how many DOS volumes are stored.
Also used to initialize the DOS 3.3 volumes

### ProROM 2025 Disk 2 - ProDOS Start Disk - ProDOS v2.4.3
Disk with an image of ProDOS v2.4.3 and the files needed to boot to DOS.3.3 or DIVERSI-DOS from ProDOS
